# Type Alias: TemplatesHomeTab

```ts
type TemplatesHomeTab = "customTemplates" | "yourStuff";
```

Identifies a tab in the Template Browser V2 experience.
Array order in [TemplatesHomeConfig.tabs](../interfaces/templates-home-config.md#tabs) determines display order.
Omit a value to hide that tab.
